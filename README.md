# Графический редактор(Лабораторная работа №1)

## Описание проекта
Данная программа представляет собой простейший графический редактор, реализованный на языке Java с использованием библиотеки JavaFX.
Программа позволяет пользователю выбирать фигуру нажатием кнопки в интерфейсе, после чего она отрисовывается на холсте.

## Иерархия классов
Все фигуры наследуются от базового абстрактного класса `Shape`, который содержит общие свойства (координаты, цвет) и абстрактный метод для отрисовки.

*Поддерживаемые фигуры:*
* Круг (`Circle`)
* Эллипс (`Ellipse`)
* Прямоугольник (`Rectangle`)
* Скругленный прямоугольник (`RoundedRectangle`)
* Квадрат (`Square`)

## Функционал
* Визуальный интерфейс с кнопками для каждой фигуры.
* Динамическое отображение выбранной фигуры на графической панели.
* Использование базовых графических примитивов JavaFX (`GraphicsContext`).

## Технологии
* Язык программирования: Java
* Графическая библиотека: JavaFX
* Система сборки: Maven / Gradle (укажи свою, если используешь)
